# chaoxing

## Commands

### assignments
- Purpose: Chaoxing assignments list
- Args:
  - `course`(optional): type: string; Filter by course name (fuzzy match)
  - `status`(optional): type: string; default: 'all'; Filter by status
  - `limit`(optional): type: int; default: 20; Maximum number of results
- Usage: `opencli chaoxing assignments [options] -f json`
