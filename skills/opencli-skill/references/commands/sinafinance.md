# sinafinance

## Commands

### news
- Purpose: Sina Finance 24/7 real-time news feed
- Args:
  - `limit`(optional): type: int; default: 20; Max results (max 50)
  - `type`(optional): type: int; default: 0; News type: 0=all, 1=A-share, 2=macro, 3=company, 4=data, 5=market, 6=international, 7=opinion, 8=central-bank, 9=other
- Usage: `opencli sinafinance news [options] -f json`
